# NotifyMeBot

NotifiMeBot is a command launcher with status notifications via Telegram message

## Installation

```powershell
git clone https://github.com/djbelyak/NotifyMeBot.git
cd NotifyMeBot
pip intsll .
```

## Usage

```powershell
notifyme --token <Telegram API token> --user_id <Telegram user id> <command>
```
